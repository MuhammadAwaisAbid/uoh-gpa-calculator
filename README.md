# University of Haripur GPA & CGPA Calculator

This web-based GPA and CGPA calculator is designed specifically for students of the University of Haripur and follows the official grading system.

## Features
- Semester GPA calculation
- CGPA calculation across multiple semesters
- Official University of Haripur grading rules
- Simple and responsive UI
